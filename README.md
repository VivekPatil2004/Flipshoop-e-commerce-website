# FlipShop - E-Commerce Website

FlipShop is a modern, responsive front-end template for an e-commerce web application. It features a clean, user-friendly interface designed with HTML and CSS, aiming to provide an engaging shopping experience through detailed product showcases.

## Features
- **Responsive Navigation:** Interactive top header and category sub-nav menus.
- **Dynamic Image Slider:** Pure CSS/HTML-based interactive header slider to highlight top offers and new arrivals out of the box.
- **Product Spotlights:** Well-organized sections for "Best Deals", "Top Offers", and "Recommended for You".
- **Product Pages:** Dedicated static template pages for varied product categories including electronics, fashion, and appliances (e.g., MacBooks, Earbuds, Fridge, Televisions).
- **Cart & Checkout Flow:** Pre-designed UI structures for Cart and Checkout experiences.
- **FontAwesome Integration:** Clean iconography integrated out-of-the-box leveraging FontAwesome 6.4.0.

## Tech Stack
- **HTML5** for semantics and structure.
- **Vanilla CSS3** for styling, fluid layouts, grid/flexbox based setups, and responsive break-points (`style.css`, `Deal.css`, `slider.css`, `offer.css`, `Rec.css`, `cart.css`, `checkout.css`, `category.css`).

## Project Structure
- `index.html`: The main landing page.
- `category.html`: The browsing page for product categories.
- `product-*.html`: Detailed product pages.
- `cart.html`: The shopping cart design.
- `checkout.html`: The checkout step UI.
- `image/`: Directory containing static media assets.

## Setup & Usage
1. Clone the repository.
2. Open `index.html` in your favorite modern web browser.
3. No build tools or Node.js environment required to run locally.
